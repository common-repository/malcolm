Bring your Malcolm! content into your Wordpress site.

== Description ==

Malcolm! makes customer self service easy with FAQs, Knowledge Base, Workflows, No-code Forms, Payments, Integrations and much more. Find out further details at https://malcolm.app.

Our Wordpress plugin makes it even simpler to bring the content you create in Malcolm! into your Wordpress site.

== Frequently Asked Questions ==

Visit [Malcolm! Support](https://support.malcolm.support "Malcolm! Support") for frequently asked questions.

== Screenshots ==

1. In MyMalcolm go to "Share > Plugins" to install and active the plugin.
2. In the embed code window toggle to "Embedding in a Wordpress site?" for Wordpress instructions.
3. In the Malcolm! Wordpress Settings paste your Instance ID.
4. Display your content directly in your Wordpress pages and posts.

== Usage ==

Insert **Inline** embeds with the following short code.

    [malcolm_inline "<Insert Embed ID>"]

Insert **Overlay** embeds with the following short code.

    [malcolm_popup "<Insert Embed ID>"]

**Widget** embeds will show automatically.
